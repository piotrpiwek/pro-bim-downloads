# PRO-BIM Downloads

Publiczne repozytorium plików do pobrania dla narzędzi PRO-BIM.

Kod aplikacji nie jest przechowywany w tym repozytorium. Repo służy tylko do hostowania lekkich instalatorów i artefaktów pobieranych ze strony PRO-BIM.

## PRO-BIM Transcriber

- `PRO-BIM-Transcriber-Setup.exe` - lekki instalator Windows.
- Instalator przy pierwszym uruchomieniu pobiera zależności lokalnie na komputer użytkownika.
